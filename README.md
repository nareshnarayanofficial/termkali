## ABOUT TOOL :

Kalimux is a bash based script which automatically installs kali linux in termux without any issue and without root. This tool works on both rooted Android device and Non-rooted Android device.

## INSTALLATION [Termux] :

```bash
apt-get update -y
apt-get upgrade -y
pkg install python -y
pkg install python2 -y
pkg install git -y
pip install lolcat
git clone https://github.com/nareshnarayanofficial/kalimux
cd $HOME
ls
cd kalimux
ls
sh kalimux.sh
```

After installation completes, the setup command will appear — copy it by long-pressing the text.

Then run:
```bash
./start-kali.sh
```

Now Kali Linux has been installed successfully in Termux without root.

## GUI SETUP (Optional) :

To use Kali Linux with a graphical interface, download the **VNC Viewer** app on your Android device.

Start Kali Linux in Termux, paste the copied command, and wait for the desktop environment to install.

After installation, run inside Kali:
```bash
vncserver-start
```

The VNC server starts on `127.0.0.1:5901`.

Open VNC Viewer, tap **+**, and create a connection to that address to access the Kali Linux GUI.

> **Note:** Do not remove Termux from the background while using Kali Linux in VNC Viewer, as VNC host files are stored in Termux.
